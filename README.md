# 🎲 随机分配抽奖工具

一个简洁美观的随机分配工具，支持人员与物品的随机匹配，适用于抽奖、分组、任务分配等场景。

## ✨ 功能特点

- 🎯 **随机分配**：将人员与物品进行随机一对一匹配
- 🕐 **实时时钟**：显示北京时间，精确到秒
- 🎨 **精美界面**：渐变色背景，流畅动画效果
- 📱 **响应式设计**：支持各种屏幕尺寸
- ⚡ **即时使用**：无需安装，打开即用

## 🚀 快速开始

### 在线使用

直接访问：[点击这里使用](https://github.com/s1313s/number-letter-lottery/)

1. 下载 `1.html` 文件
2. 双击打开即可使用

## 📖 使用说明

1. **输入人员名单**：在第一个输入框中输入参与人员，用逗号分隔
   - 例如：`张三,李四,王五,赵六`

2. **输入物品名单**：在第二个输入框中输入奖品或任务，用逗号分隔
   - 例如：`一等奖,二等奖,三等奖,四等奖`

3. **开始抽奖**：点击"开始随机分配"按钮

4. **查看结果**：系统会随机将物品分配给人员，并显示匹配结果

## 💡 使用场景

- 🎁 **年会抽奖**：随机分配奖品给员工
- 👥 **分组活动**：将成员随机分配到不同小组
- 📋 **任务分配**：随机分配工作任务
- 🎮 **游戏抽签**：游戏中的角色或位置分配
- 🍽️ **聚餐AA**：随机分配付款顺序

## ⚠️ 注意事项

- 人员数量必须与物品数量相等
- 多个名称之间用**英文逗号**分隔
- 支持中文、英文、数字等各种字符
